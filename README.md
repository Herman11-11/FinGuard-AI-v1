# FinGuard-AI

A digital fingerprint system for land documents in Tanzania.

## What it does
- Protects land titles from forgery
- Gives each document a unique digital fingerprint
- Helps Ministry of Lands verify documents instantly

## Built for
Ministry of Lands, Tanzania 🇹🇿

## Features
- SHA-256 hashing (detects tampering)
- GAN fingerprints (unique to each document)
- 3-person authentication (security)
- Mobile app for field officers
- Swahili/English support
